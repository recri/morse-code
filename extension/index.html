Morse Code

Morse code is the original digital communications medium.

Text Output

This output page will let you enter text and hear it translated into morse code.

Key Input

This input page will let you key in morse code with a straight key or iambic paddle.

Audio Decoding

This decoding page will listen to morse code over your computer's microphone and decode it.

Training

This training game page will play morse code patterns for you to repeat back.

WebRTC

This WebRTC page will allow you to practice morse code with other people over the internet.
